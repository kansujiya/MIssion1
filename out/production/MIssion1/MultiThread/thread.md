"Process" : Process is instance of program that being executed. Its has own resource like memory, thread etc.

compilation javac Test.java => generated byte code that can be executed by jvm.
Execution : java test => jvm start new process

java -Xms256m -Xmx2g test

-Xms - initial heap size
-Xmx - max heap size

Thread : light weight process OR The Smallest sequence of instruction executed by CPU

javac test.java => byte code => java test => new process
